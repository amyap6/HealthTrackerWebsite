# HealthTrackerWebsite

This is a web application of a health tracker written in Python, Django and HTML.
With the health tracker you can make an account, track your calorie and nutrition intake, set weight loss/gain goals and join groups.

Dependencies to run this website:
1. Make sure you have python installed on your machine - at least version 3.6
2. Make sure you have django installed on your machine - at least version 3.0.3

To run the website:
1. The easiest way is to fork this repository and run the project in your favourite IDE - we used PyCharm.
